# AI Life Engine local public site

This is a minimal static deployment package for personal cross-network access.

Included:
- `index.html`
- `assets/index-B7-XkV0u.js`
- `assets/index-Bz31YfrW.css`

Not included:
- Local saves
- API logs
- Screenshots
- Backup bundles
- Built-in public API keys

The bundled default API key strings were replaced with `PUBLIC_BUILD_API_KEY_REMOVED`.
Configure your own API endpoint and key in the site settings after opening the page.

## GitHub Pages

This repository includes a GitHub Actions workflow that deploys the repository root
to GitHub Pages on every push to `main`.
